## 用户端使用接口

同一前缀:[](/user)

| 网络尾缀  | 请求方法 | 参数格式                                           | 回送数据格式 | 备注     |
| --------- | -------- | -------------------------------------------------- | ------------ | -------- |
| /register | POST     | 四个字符串字段，username、password、phone、address |              | 用户登录 |
| /login    | POST     | 两个字符串字段，username、password                 |              | 用户注册 |
